# ☁️ NC台账管理系统 - 云端部署指南

## 方案说明

把系统部署到云端（Render.com），获得一个**永久可访问的网址**，不受本地网络和电脑开关机影响。

---

## 🚀 部署步骤（免费）

### 第一步：注册Render账号

1. 打开 https://render.com
2. 点击「Sign Up」用 **GitHub** 账号登录
3. 完成邮箱验证

### 第二步：创建Web服务

1. 登录后，点击「New」→ 「Web Service」
2. 选择「Build and deploy from a Git repo」（如果没有GitHub，点击下面链接用zip部署）
3. 推荐用 GitHub：https://github.com/new 创建新仓库，命名为 `nc-account-system`
4. 把本文件夹内容上传到GitHub仓库

**或者更简单 - 用ZIP直接部署：**

1. 打开 https://render.com/dashboard
2. 点击「New」→ 「Static Site」
3. 拖入本文件夹的zip文件

### 第三步：配置服务

选择 **Free** 套餐：
- 免费额度：每月750小时
- 休眠后唤醒需30秒
- 完全够用！

---

## 📋 上传到GitHub方法

1. 打开 https://github.com/new
2. 仓库名：`nc-account-system`（设为Public）
3. 创建后按提示上传文件：
   ```bash
   git init
   git add .
   git commit -m "NC台账系统"
   git remote add origin 你的仓库地址
   git push -u origin main
   ```

4. 在Render选择这个仓库，填入：
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn cloud_app:app`

---

## ✅ 部署完成

Render会给你一个类似这样的网址：
```
https://nc-account-system.onrender.com
```

这个网址**永久有效**，任何网络都能打开！

- 添加/编辑数据自动保存
- 数据存储在云端

## ⚠️ 注意事项

1. **免费版会休眠**：超过15分钟无访问会休眠，再访问需等待30秒唤醒
2. **唤醒后数据在**：数据存储在云端，不会丢失

如需24小时在线，可升级付费版（$7/月）或保持本地+穿透方案。
